SI units of information

$$1 bit = 1.0 bit$$

$$1 bps = 1.0 bps$$

$$8 bit = 1.0 byte$$

$$1 byte / s = 8.0 bps$$

$$1 kbit = 125.0 byte$$ $$1 kB = 1000.0 byte$$

$$1 kbit / s = 1000.0 bps$$ $$1 kB / s = 8000.0 bps$$

==== MB ====

$$R1 := 1 Mbit$$ $$R2 := 1 MB$$ $$n := 2$$

$$R1 = 1000.0 kbit$$ $$R1 = 125.0 kB$$ $$R2 = 1000.0 kB$$

$${1000}^{n} \cdot bit - R1 = 0.0 bit$$

$${1000}^{n} \cdot byte - R2 = 0.0 byte$$

$$R1 / s = 1000.0 kbps$$ $$R2 / s = 1000.0 kBps$$
